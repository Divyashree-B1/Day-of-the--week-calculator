# Day-of-the--week-calculator
“A program that finds the day of the week for any given date.”

## Description
The Day of the Week Calculator is a simple program that calculates the day of the week for any given date.  
It helps users quickly find out whether a specific date falls on Monday, Tuesday, etc.

## Features
- Calculates the day of the week for any date
- User-friendly interface 
- Handles leap years accurately
- Works for past, present, and future dates

## Technologies Used
- Python
- Tkinter – Built-in Python library for creating GUI (Graphical User Interface)
- Built-in 'datetime' module.
